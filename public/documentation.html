Documentation for movie api

Description of Endpoint Return Endpoint URL Method Type Format of Request Data Format of Response Data
Return a list of ALL movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about the requested movie ex: { "title": "Inception", "director": "Christopher Nolan", "genre": "Sci-Fi" }
Return data about a single movie's genre by title /movies/:title/genre GET None A JSON object holding data about the genre of a specific movie ex: "Sci-Fi"
Return data about a single movie's director by title /movies/:title/director GET None A JSON object holding data about the director of a specific movie ex: "Christopher Nolan"
Return the documentation page /documentation GET None A html page holding the documentation page of this app
Create new user /users POST A JSON object holding data from the user to be added ex: { "fullname": "Joey", "email": "joey@mail.com" } A JSON object holding data from the user that has been added plus uuid ex: { "fullname": "Joey", "email": "joey@mail.com", "id": "200f45b2-d144-49a6-9252-863c0fec99a9" }
Update user name/email by uuid /users/:id PUT A JSON object holding data from the user to be updated ex: { "fullname": "Joana Doe", "email": "joanadoe@mail.com" } A JSON object holding data from the user that has been updated ex: { "user": "2", "fullname": "Joana Doe", "email": "joanadoe@mail.com", "favMovies": [ { "title": "Inception", "director": "Christopher Nolan", "genre": "Sci-Fi" } ] }
Update users favMovies by title and user id /users/:id/:title POST A JSON object holding data from the user about favMovies to be updated ex: { "director": "Michael Bay", "genre": "Horror" } A message returning if the users favMovies have been successfully updated with the updated favMovies logged on the console ex: [{"title":"Inception","director":"Christopher Nolan","genre":"Sci-Fi"},{"title":"Insidious","director":"Michael Bay","genre":"Horror"}]
Delete a user's favMovie /users/:id/:title DELETE None A message returning if the users favMovies have been successfully updated with the updated favMovies logged on the console ex: [{"title":"Inception","director":"Christopher Nolan","genre":"Sci-Fi"}]
Delete a user /users/:id/ DELETE None A message returning if the user was successfully deleted
